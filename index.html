슬라이드 1: Gemini CLI 소개

안녕하세요! 저는 Google에서 개발한 대규모 언어 모델인 Gemini CLI입니다.

사용자의 소프트웨어 엔지니어링 작업을 돕기 위해 설계되었습니다.

슬라이드 2: 저의 역할

  • 코드 작성, 수정, 리팩토링
  • 파일 및 디렉토리 관리
  • 쉘 명령어 실행
  • 프로젝트 설정 및 빌드
  • 문제 해결 및 디버깅 지원

슬라이드 3: 저의 능력

  • 다양한 프로그래밍 언어 이해 및 생성
  • 기존 코드베이스 분석 및 컨벤션 준수
  • 새로운 애플리케이션 프로토타입 개발
  • 보안 및 안전 지침 준수

슬라이드 4: 사용자와의 상호작용

저는 CLI 환경에서 텍스트 기반으로 소통합니다.

명령어를 통해 파일 시스템에 접근하고, 코드를 실행하며, 필요한 정보를 얻습니다.

사용자의 지시를 명확히 이해하고, 필요한 경우 추가 정보를 요청합니다.

슬라이드 5: 저의 한계

  • 시각적 인터페이스 직접 조작 불가
  • 물리적 세계와의 상호작용 불가
  • 복잡한 인간 감정 및 의도 완벽 이해의 어려움
  • 파워포인트 파일과 같은 특정 애플리케이션 파일 직접 생성 불가

슬라이드 6: 저를 활용하는 방법

명확하고 구체적인 지시를 내려주세요.

필요한 경우 파일 경로, 코드 스니펫 등 충분한 컨텍스트를 제공해주세요.

궁금한 점이나 필요한 작업이 있다면 언제든지 질문해주세요!
